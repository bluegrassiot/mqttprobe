# mTLS lab certificates

Generated by:

```bash
python scripts/generate-mtls-certs.py
```

Do not commit private keys or PFX files. This directory is gitignored except this README.

| File | Purpose |
|------|---------|
| `ca.crt` / `ca.key` | Lab CA |
| `server.crt` / `server.key` | Mosquitto TLS identity |
| `client.crt` / `client.key` | Client PEM for MQTTProbe |
| `client.pfx` | Client PFX for MQTTProbe (default password `changeme`) |

After generating certs:

```bash
docker compose -f docker-compose.mtls.yml up -d
```

Import `client.pfx` (or PEM pair) in MQTTProbe. Connect to `127.0.0.1:8883` with TLS and **Allow untrusted certificate** enabled for the lab server cert.
