# Security Policy

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Email security reports to: info@bluegrassiot.com

Include a description of the issue, reproduction steps, and potential impact.
You can expect an acknowledgement within 48 hours and a resolution timeline
within 14 days for confirmed issues.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |
| < 1.0   | No        |
